# 《企业AI平台与Agent应用落地手册》

## 项目定位

本项目不是普通 AI 学习笔记，而是一套面向企业软件工程师、架构师、技术负责人和数字化团队的企业 AI 平台与 Agent 应用落地手册。

本手册覆盖企业 AI 平台、AI 知识库、RAG、Agent、MCP、Workflow、Multi-Agent、企业业务场景、私有化部署、治理运营与持续交付，目标是把 AI 技术能力转化为企业可建设、可集成、可运维、可治理的系统化方案。

平台基础能力范围包括组织、部门、人员、账号、岗位、角色、菜单权限、功能权限、数据权限、接口权限、工具调用权限、Agent 使用权限、审批权限、审计日志和操作留痕。这些能力是所有 Agent、RAG、Workflow、MCP 和业务系统集成的统一底座。

## 核心目标

1. 系统梳理企业 AI 平台、AI 知识库、RAG、Agent、MCP、Workflow、Multi-Agent、A2A 等核心技术与工程边界。
2. 结合 HR、薪酬、ERP、OA、Office、制造、财务、用户、订单、结算等真实企业业务场景，形成可落地的架构方案、流程设计和案例模板。
3. 以 Java / Spring Boot / Vue / Python 技术栈为主，逐步构建可演进的企业级 AI 平台与 Agent 应用 Demo。
4. 建立企业 AI 私有化部署、组织人员、菜单权限、数据权限、工具调用权限、规则引擎、性能设计、可观测性、成本控制和治理运营方法。
5. 所有文档优先采用 Markdown 管理，所有图形保留 Mermaid 或 Draw.io 源文件。
6. 本地 Markdown 是唯一数据源，飞书、PDF、网站、Word 都只是后续发布形式。
