# Spring Boot示例

版本：v1.0  
更新时间：2026-06-29  
适用对象：企业软件工程师 / 架构师 / 技术负责人  

## 1. 本章核心结论

本目录用于存放企业 AI Agent 后端服务示例，重点覆盖 API、权限、工具适配和审计日志。

## 2. 背景与问题

企业 Agent 后端需要与已有 Java 体系、统一认证和业务系统集成。

## 3. 核心概念

Controller、Service、ToolAdapter、ModelGateway、AuditLog。

## 4. 应用架构

Spring Boot 作为企业服务底座，调用模型网关、RAG 服务和业务系统工具。

## 5. 工作流程

接收任务请求，校验权限，执行 Agent 编排，返回结果并写入日志。

## 6. 企业案例

实现 HR Agent 的政策问答和员工信息查询接口。

## 7. 技术实现建议

先定义接口契约，再补充具体框架代码。

## 8. 常见问题

问：是否直接在后端拼接所有提示词？  
答：可以先这样做，后续应抽象为模板管理能力。

## 9. 后续延伸

补充 Maven 项目结构和示例接口。

## 10. Mock API 契约

v1.1 样板深化阶段已新增 [mock-api-contracts.md](mock-api-contracts.md)，用于描述用户、订单、结算、财务、发票、凭证、对账、审批和审计相关 Mock API。
