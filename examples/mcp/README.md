# MCP示例

版本：v1.0  
更新时间：2026-06-29  
适用对象：企业软件工程师 / 架构师 / 技术负责人  

## 1. 本章核心结论

本目录用于存放 MCP Server 和工具定义示例，验证 Agent 与企业系统的标准化连接方式。

## 2. 背景与问题

企业系统能力需要以结构化工具方式暴露给 Agent，避免重复适配。

## 3. 核心概念

MCP Server、Tool Schema、Resource、Prompt。

## 4. 应用架构

MCP 示例服务封装 HR、OA 或 ERP 的只读查询能力。

## 5. 工作流程

定义工具，启动服务，Agent 发现并调用工具，服务返回结构化结果。

## 6. 企业案例

HR MCP Server 提供组织查询和员工基础信息查询工具。

## 7. 技术实现建议

每个工具必须写清楚参数、返回结构、权限和错误码。

## 8. 常见问题

问：MCP 示例是否可以不做权限？  
答：本地实验可以简化，但文档必须标注生产环境权限要求。

## 9. 后续延伸
