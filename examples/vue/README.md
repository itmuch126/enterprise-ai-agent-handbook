# Vue示例

版本：v1.0  
更新时间：2026-06-29  
适用对象：企业软件工程师 / 架构师 / 技术负责人  

## 1. 本章核心结论

本目录用于存放 Agent 工作台前端示例，重点展示会话、任务状态、引用来源和人工确认。

## 2. 背景与问题

企业用户需要透明看到 Agent 执行过程，而不只是看到一段最终回答。

## 3. 核心概念

Agent 切换、会话流、任务轨迹、来源引用、反馈入口。

## 4. 应用架构

Vue 前端通过 API 或流式接口连接 Spring Boot 后端。

## 5. 工作流程

选择 Agent，输入任务，查看执行步骤，确认关键操作，提交反馈。

## 6. 企业案例

HR Agent 工作台展示制度引用和流程发起确认。

## 7. 技术实现建议

界面应按任务场景组织，不要只做通用聊天框。

## 8. 常见问题

问：是否需要管理端？  
答：需要，后续用于管理 Agent、工具和知识库。

## 9. 后续延伸

补充页面原型和组件结构。
