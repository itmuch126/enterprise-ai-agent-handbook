## 6. v1.1 样板深化阶段说明

当前项目基线版本仍保持为“企业AI平台与Agent应用落地手册 v1.0”。v1.1 阶段重点为业务 Agent 样板深化，优先补充字段样例、接口样例、规则编号、权限矩阵、异常处理、测试用例、审批确认、审计字段、性能指标、缓存策略和降级策略。

v1.1 阶段新增样例均为“示例，需结合企业实际系统确认”，不代表真实接口地址、真实客户数据、具体厂商能力或固定业务规则。

v1.1 样板深化第二轮重点包括：订单 Agent 样板深化、业务协同/财务/订单/结算领域对象模型图、关键业务时序图、Spring Boot Mock API 契约和样例 JSON。

v1.1 阶段后续图示深化应遵循统一画图规范：每个重要章节至少配套静态架构图、动态 Workflow 图、时序图、部署架构图中的若干种。图示规范将服务于后续汇报材料、架构设计、开发实现、运维实施和业务理解。

v1.1 平台级图示补充阶段已新增企业AI平台能力地图、企业AI平台部署架构图、MCP 工具注册流程图、MCP 工具调用时序图、Multi-Agent 协作架构图和 Multi-Agent 任务协同时序图。后续建议继续补充 RAG 知识数据流图、财务数据流图、订单数据流图和结算数据流图。

v1.1 数据流图与业务链路图补充阶段已补充企业知识库 RAG 知识数据流图、订单 Agent 数据流图和结算金额计算链路图，并同步修复领域对象模型图的 Mermaid 源文件与 Markdown 正文嵌入内容一致性。后续继续补充财务数据流图、用户数据流图、订单状态机图和汇报用业务架构图。

v1.2 总控架构收敛阶段用于统一表达现有 v1.0/v1.1 成果，不改变项目基线版本号。该阶段新增企业AI平台总控架构图，形成“单总控图 + 分层图”的架构表达体系；旧企业AI平台总体架构图降级为模块视角图，Agent、MCP、RAG、Workflow、数据流图、时序图和领域模型图均作为专题图向总控架构图对齐。
