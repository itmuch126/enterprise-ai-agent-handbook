# 招聘与员工服务Agent

版本：v1.0  
更新时间：2026-06-29  
适用对象：企业软件工程师 / 架构师 / 技术负责人  

## 1. 本章核心结论

招聘和员工服务是 HR Agent 最容易产生价值的两个方向，前者提升 HR 效率，后者提升员工体验。

## 2. 背景与问题

招聘流程信息分散，员工服务咨询重复，HR 经常被事务性问题占用。

## 3. 核心概念

- 招聘 Agent：职位描述生成、简历初筛、面试安排。
- 员工服务 Agent：制度问答、流程导航、材料生成。
- 人工复核：关键筛选和员工敏感问题需要人工确认。

## 4. 应用架构

招聘 Agent 接入 ATS 或招聘系统，员工服务 Agent 接入知识库、OA 和 HR 系统。

## 5. 工作流程

Agent 收集需求，检索规则，调用业务系统，生成建议或发起流程。

## 6. 企业案例

候选人简历进入后，招聘 Agent 提取核心信息并生成初筛建议给 HR 复核。

## 7. 技术实现建议

招聘建议必须保留可解释理由，避免黑箱筛选和偏见风险。

## 8. 常见问题

问：招聘 Agent 能否自动淘汰候选人？  
答：不建议。应作为辅助筛选工具，由 HR 做最终决定。
